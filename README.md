# Fleet Tracker Database & Web Interface

A database-driven fleet tracking system designed to store and manage GPS telemetry data from vehicle tracking devices.

## Technologies
- SQL
- JavaScript
- HTML
- CSS
- Database Design

## Overview
This project implements a relational database system for storing and managing GPS tracker data from fleet vehicles. The system allows users to input, organize, and query tracker information through a web interface.

The database structure is designed to efficiently handle telemetry data such as vehicle identifiers.

## Features
- Store GPS telemetry data from vehicle trackers
- Manage tracker and vehicle information
- Query and organize fleet tracking records
- Web interface for data entry and management

## System Components
- Relational SQL database for storing telemetry data
- Web interface built with JavaScript, HTML, and CSS
- Structured database tables for efficient querying and data management

## Purpose
The goal of this project is to demonstrate database design and web interface integration for a fleet monitoring system capable of managing real-time vehicle tracking data.
